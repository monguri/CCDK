VCE Package Contents

This package contains the following directories and files.

  • bin/

    Includes execution file groups for Linux and Ruby script groups. Compiled sample programs are also included in this directory.

  • include/

    Includes C language header files that can be used in Linux, Win32, and “PlayStation(R)2”.

  • docs/

    Includes VCE API manuals and manuals for groups of various tools.

  • lib/

    Includes library binaries for Linux.

  • samplecode/

    Most of the sample codes are tested in Linux. With small changes (such as #include), they can be used in Windows as well.

  • autotest/

    A string of test programs used for VCE development. Due to the strict operating system restrictions and experimental codes included during development, many of the tests cannot be performed in a user environment. The codes can be used as samples or reference.
